How to use the PowerPoint Narration Generator

📋 Overview

This tool adds spoken narration to your PowerPoint presentation using Azure AI Text-to-Speech. You provide a script and a presentation, and the tool matches each script section to a slide, embeds the audio automatically, and lets you download the result. You can export as a narrated .pptx or as a self-contained .mp4 video.

🔢 Step 1 — Upload Files

  1. Narration Script — upload a .docx Word document or a .pptx file containing your narration text.
    • For Word documents: use Heading 1 paragraphs (or paragraphs that start with "Slide N") as slide separators. The text under each heading becomes the audio for that slide.
    • For PowerPoint scripts: slide text is read directly from each slide in order.
    • The document must not be password-protected.
  2. PowerPoint Presentation — upload the .pptx file you want narration added to.
  3. Language & Voice — choose from 30+ neural voices across multiple languages.
  4. Click Next: Preview Slides → when both files are ready.

✦ AI Mode (optional)

Toggle Generate presentation with AI on Step 1 to skip uploading a PowerPoint entirely. The AI reads your script and builds a complete slide deck — including slide layouts and generated images — then narrates it. No .pptx upload needed.

🔢 Step 2 — Review Slide Mapping

The tool automatically matches each section of your script to a slide. Review the table and use the dropdown in each row to correct any mismatches. Script sections with no matching slide are left as unmapped (shown with a warning banner) — you can reassign them using the dropdowns.

🔢 Step 3 — Generate & Download

Click Generate Narrated PPTX →. The tool sends each slide's narration to Azure Neural TTS, embeds the resulting audio clip directly into the matching slide, and packages the final file. A progress bar shows real-time status.

When complete, two download options appear:

  • Download PPTX — the presentation with embedded audio. Open in PowerPoint and run the slideshow — audio plays automatically on each slide transition.
  • Download as Video (MP4) — a video where each slide is displayed with its narration playing. Ideal for sharing without requiring PowerPoint.

🔢 Step 4 — Quality Check

After downloading, run an optional quality check. The tool transcribes each embedded audio clip with Azure Speech-to-Text and scores it against your original script using AI. Results show a confidence percentage and any critical or minor discrepancies per slide.

The script and voice fields are pre-filled from Step 1 — just click Run Quality Check →.

💡 Tips

  • Keep script sections concise — very long passages may hit TTS character limits.
  • If slide counts don't match, use the Step 2 dropdowns to reassign script sections manually.
  • MP4 export takes a minute or two for large decks — the progress bar keeps you informed.
  • Click Start Over to reset everything and begin with a new presentation.
