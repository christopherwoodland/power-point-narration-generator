# Support

## How to Get Help

This is an **internal Microsoft tool** and is not a supported Microsoft product. Support is provided on a best-effort basis by the maintaining team.

### GitHub Issues (preferred)

For bug reports, feature requests, and questions, please [open a GitHub issue](../../issues).

Before opening a new issue:
- Search existing issues to avoid duplicates.
- Include relevant details such as OS, browser, .NET version, and steps to reproduce.

### Security Issues

For security vulnerabilities, do **not** open a public issue. Follow the responsible disclosure process described in [SECURITY.md](SECURITY.md).

## What Is Not Supported

- This project is not covered by any Microsoft support plan or SLA.
- It is not a generally available Microsoft product or service.
- For questions about underlying Azure services (Azure Speech, Azure OpenAI, Azure Translator), refer to official [Azure documentation](https://learn.microsoft.com/azure) and [Azure support channels](https://azure.microsoft.com/support/options/).
